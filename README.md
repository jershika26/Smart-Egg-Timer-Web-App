Smart Egg Timer Web Application

The Smart Egg Timer is a modern and interactive web application that helps users cook eggs perfectly using preset timers for soft, medium, and hard-boiled options. Built using HTML, CSS, and JavaScript, the app offers a simple and responsive interface suitable for all devices.

Features:
Timer options for soft, medium, and hard-boiled eggs
Accurate countdown functionality
Alert/notification when the timer completes
Clean, minimal, and responsive user interface
Works on both mobile and desktop browsers

Technologies Used:
HTML5 for structure
CSS3 for styling and layout
JavaScript (ES6) for timer logic and interactivity

 Project Structure:
The-smart-egg-timer-web-app/
│── index.html
│── style.css
│── script.js
└── assets/
     ├── images/
     └── sounds/

Usage:

Open the application in your browser.
Select the preferred egg type:
Soft
Medium
Hard
Start the timer.
Wait for the alert when the timer completes.
